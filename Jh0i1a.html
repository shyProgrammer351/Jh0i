문제

재귀함수를 이용하여, 정수를 입력받아 0부터 입력값까지의 합계 Σ를 계산하는 문제입니다 이를 해결하는 방법을 아래 참고자료에서 한줄씩 복붙해서 설명하세요 
숫자를 입력하세요: 3
Σ 3 = 3 + 2 + 1 + 0 = 6 


숫자를 입력하세요: 5
Σ 5 = 5 + 4 + 3 + 2 + 1 + 0 = 15 


숫자를 입력하세요: 0
Σ 0 = 0 = 0
