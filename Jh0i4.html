문제

하노이 탑에 첫번째 기둥에 있는 디스크를 세번째 기둥으로 모두 옮기는 문제입니다 이를 해결하는 다음 프로그램을 해석하세요 
 				
Move one disk from 1 to 3
Move one disk from 1 to 2
Move one disk from 3 to 2
Move one disk from 1 to 3
Move one disk from 2 to 1
Move one disk from 2 to 3
Move one disk from 1 to 3 


 	  				
알고리즘
 				
 프로그램시작 
    하노이 타워 생성 
    3개의 원판을 1(출발), 2(임시), 3(도착) 기둥을 이용해 이동 
 프로그램종료 
 
 원판 이동 함수 시작 
    N-1개 원판을 출발기둥에서 임시기둥으로 모두 이동 
    가장 큰 원판을 출발기둥에서 도착기둥으로 이동 
    N-1개 원판을 임시기둥에서 도착기둥으로 모두 이동 
 		 				
프로그램 코드 
  				
	// 파일명 : ./Chapter20/TowerOfHanoi_Main.java

	class TowerOfHanoi

	{

		// 원판 이동 함수 시작 



1		public void moveTower( int numDisks, int start, int temp, int end ) {



2			if ( numDisks == 1 )



3				moveOneDisk( start, end );



4			else {

		  		// N-1개 원판을 출발기둥에서 임시기둥으로 모두 이동 



5				moveTower( numDisks-1, start, end, temp );

				// 가장 큰 원판을 출발기둥에서 도착기둥으로 이동 



6				moveOneDisk( start, end );

				// N-1개 원판을 임시기둥에서 도착기둥으로 모두 이동 



7				moveTower( numDisks-1, temp, start, end );

			}



8		}



9		private void moveOneDisk( int start, int end ) {



10			System.out.println( "Move one disk from " + start + " to " + end );

		}

	}

		public class TowerOfHanoi_Main

	{

		//프로그램시작 



11		public static void main( String[] args )

		{

			//하노이 타워 생성 



12			TowerOfHanoi towers = new TowerOfHanoi();

			// 3개의 원판을 1(출발), 2(임시), 3(도착) 기둥을 이용해 이동 



13			towers.moveTower( 3, 1, 2, 3 );

			//프로그램종료 



14			return;

		}

	}
