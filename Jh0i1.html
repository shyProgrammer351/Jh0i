숫자를 입력하세요: 3
Σ 3 = 3 + 2 + 1 + 0 = 6 


숫자를 입력하세요: 5
Σ 5 = 5 + 4 + 3 + 2 + 1 + 0 = 15 


숫자를 입력하세요: 0
Σ 0 = 0 = 0
