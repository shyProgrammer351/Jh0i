20. 재귀메서드

재귀메서드   메서드 내에서 자기자신을 호출하는 메서드이며, 메서드를 호출할 때마다 별도의 메모리 공간이 할당
문제를 직관적으로 해결할 수 있지만, 반복문에 비해 메모리 공간이 더 필요하고 이에 따른 속도가 저하
예) n! = n * (n-1)! : !(factorial) 메서드에서 !(factorial) 메서드를 다시 호출, 5! = 5 * ( 4 * 3 * 2 * 1 ) = 5 * 4!

주의사항 무한호출을 예방하기 위해서, 재귀메서드 종료 조건을 반드시 재귀메서드 호출보다 위쪽에 명시

재귀메서드 : n! = n * (n-1)!     단, n > 1, 1!=1, 0!=1 반복 : n! = n * (n-1) * (n-2) * ... * 3 * 2 * 1
import java.util.Scanner;
public class Factorial
{
  public static int factorial( int num ) {
    if ( num <= 1 ) {
      System.out.print ( "1 = " );
      return 1;
    }
    else {
      System.out.print ( num + " * " );
      return num = num * factorial( num - 1 );
    }
  }
  public static void main( String[] args ) {
    Scanner scan = new Scanner( System.in );
    int num = 0, result = 0;
    System.out.print( "숫자를 입력하세요: " );
    num = scan.nextInt();
    System.out.print( num + "! = " );
    result = factorial( num );
 
 
 
    System.out.print ( result );
    scan.close();
  }
